# CountryApp

## Описание
Консольное приложение на C для работы с данными о странах и регионах. Поддерживает:
- Аутентификацию пользователей
- Добавление/удаление стран и регионов
- Просмотр статистики
- Хранение данных в SQLite

## Установка
brew install sqlite3 gcc            # macOS

# Сборка
git clone https://github.com/fpmi-tp2025/tpmp-gr12-lab4-hemrahudayberdiyev1

cd tpmp -gr12-lab4-hemrahudayberdiyev1
make

## Использование

./bin/country_app


**Основные функции:**

1. Добавить страну
2. Удалить страну
3. Добавить регион
4. Удалить регион
5. Просмотреть список
6. Статистика
0. Выход

**Пример работы:**

Логин: admin

Пароль: admin123

Выберите действие: 1

Название: Беларусь

Столица: Минск

Население: 9255000

Площадь: 207600

Страна добавлена!
